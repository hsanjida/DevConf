# 🚀 DevConf 2026 - Conference Landing Page

A responsive conference landing page built using **HTML5** and **CSS3**. This project was created as a practice exercise to strengthen semantic HTML, CSS Flexbox, CSS Grid, and responsive web design skills.

---

## 📌 Project Overview

DevConf 2026 is a fictional technology conference landing page designed to showcase a modern event website. The page provides essential conference information including speakers, ticket pricing, and event registration while maintaining a clean and professional user interface.

---

## ✨ Features

- Responsive navigation bar
- Hero section with background image and call-to-action button
- Featured speakers section using CSS Grid
- Pricing cards with multiple ticket options
- Footer with social media links
- Clean and modern UI design
- Semantic HTML5 structure
- Responsive layout using Flexbox and Grid
